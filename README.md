## Getting Started

### TO run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### To run tests:

```bash
npm run test
# or
yarn test
# or
pnpm test
# or
bun test
```
